# Hospital-Record-Analysis-Dashboard
Hospital Record Analysis Dashboard

This dashboard analyzes 403 hospital visits (2019) to evaluate patient outcomes, risk levels, and mortality trends.

 Key Metrics

Total Visits: 403

Deaths: 69 (17%)

Survivals: 334 (83%)

Risk Distribution: Low (55%) | High (45%)

 Key Insights

Survival rate remains high despite a significant proportion of high-risk patients.

Monthly death trends show noticeable fluctuations, with a peak in August.

Death growth rate varies across the year, indicating seasonal and operational impact.

Cumulative deaths increase steadily mid-year, then slow toward year-end.

 Summary

The dashboard provides clear visibility into hospital performance, highlights mortality patterns over time, and supports data-driven healthcare decision-making through intuitive KPIs and visual storytelling.
